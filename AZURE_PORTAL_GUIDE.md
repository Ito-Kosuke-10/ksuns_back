# Azure Portalでのクエリエディターの開き方

## 📍 場所の見つけ方

現在、Azure Database for MySQLのプロパティページが開いています。

### 方法1: 左側のメニューから開く（推奨）

1. **左側のナビゲーションメニュー**を確認
2. **「データベース」** セクションを探す
   - 「設定」セクションの下にあります
   - まだ展開されていない場合は、クリックして展開
3. **「クエリエディター」** または **「ワークベンチ」** をクリック

### 方法2: 検索バーを使う

1. ページ上部の**検索バー**をクリック
   - 「リソース、サービス、ドキュメントの検索」と表示されている部分
2. **「クエリエディター」** と入力
3. 検索結果から **「クエリエディター」** を選択

### 方法3: 直接URLで開く

ブラウザのアドレスバーで、現在のURLの最後に `/queryEditor` を追加：

```
https://portal.azure.com/.../gen10-mysql-dev-01/queryEditor
```

---

## 🔍 見つからない場合

### 「データベース」セクションが見つからない場合

1. 左側のメニューを下にスクロール
2. **「設定」** セクションの下を確認
3. もしくは、ページ上部の**「接続」**ボタンをクリック
   - 接続情報が表示されるページが開きます
   - そこから「クエリエディター」へのリンクがある場合があります

### 別の方法：MySQL Workbenchを使う

Azure Portalのクエリエディターが見つからない場合は、**MySQL Workbench**を使うこともできます：

1. MySQL Workbenchをインストール（まだの場合）
2. 新しい接続を作成：
   - Hostname: `gen10-mysql-dev-01.mysql.database.azure.com`
   - Username: `students`
   - Password: （Azure Portalで確認）
   - Port: `3306`
3. 接続後、`create_tables.sql` を開いて実行

---

## 📝 クエリエディターでSQLを実行する手順

1. **クエリエディター**が開いたら
2. ログイン情報を入力：
   - ユーザー名: `students`（管理者のログイン）
   - パスワード: （Azure Portalで設定したパスワード）
3. **`create_tables.sql`** の内容をコピー＆ペースト
4. **実行**ボタン（▶️ または「実行」）をクリック
5. 成功メッセージが表示されれば完了！

---

## ⚠️ エラーが出た場合

- **「アクセスが拒否されました」**: ファイアウォール設定を確認
  - 左側メニューの「ネットワーク」を開く
  - 現在のIPアドレスを許可する
- **「テーブルが既に存在します」**: 問題ありません。既に作成済みです。
